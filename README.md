# Sistemas CLI em Python

Coleção de sistemas de gerenciamento em linha de comando desenvolvidos em Python para prática de lógica de programação e estruturas de dados.

## Projetos incluídos

###  Cadastro de Produtos
Sistema CRUD de produtos com ID, nome, preço e quantidade.

Funcionalidades:
- Criar produtos
- Buscar por ID
- Atualizar dados
- Excluir produtos

---

###  Lista de Tarefas
Sistema de gerenciamento de tarefas com identificação por ID.

Funcionalidades:
- Adicionar tarefas
- Visualizar tarefas
- Atualizar tarefas
- Excluir tarefas

---

###  Cadastro de Veículos
Sistema de cadastro e consulta de veículos com salvamento em arquivo.

Funcionalidades:
- Cadastro de veículos
- Listagem
- Busca
- Exportação para arquivo

---

###  Cadastro de Alunos
Sistema de cadastro de alunos com cálculo de média.

Funcionalidades:
- Cadastro de alunos
- Registro de notas
- Cálculo de média
- Exportação para arquivo

---

## Tecnologias utilizadas
- Python
- Listas e dicionários
- Funções
- Estruturas condicionais e loops
- Manipulação de arquivos

## Objetivo
Praticar desenvolvimento de aplicações CLI e operações CRUD em Python.
